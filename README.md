# Blog de Estudos

Um pequeno blog de estudos para salvar artigos, revisar o que você aprendeu e acompanhar os livros que você leu.

## Funcionalidades

* **Criação de Artigos:** Crie artigos de estudo com um título e uma área de conteúdo ampla.
* **Capacidade:** Escreva até 1.000 linhas por artigo.
* **Mídia:** Faça upload de imagens e outros arquivos de mídia para anexar aos artigos.
* **Destaques:** Marque artigos selecionados como "em destaque" e visualize-os na página inicial.
* **Navegação:** Navegue por todos os artigos publicados em uma lista de artigos dedicada.
* **Leitura:** Abra artigos em uma visualização de leitura para ver o conteúdo completo e as mídias anexadas.
* **Multi-idioma:** Alternar a interface entre Inglês e Português (com salvamento de preferência para visitas futuras).
* **Gerenciamento de Livros:** Aba dedicada para adicionar livros lidos, armazenando título, autor, nota (avaliação) e observações pessoais.
* **Lista de Leitura:** Visualize sua biblioteca pessoal com as respectivas avaliações.
* **Manutenção:** Opção para excluir artigos ou livros que não forem mais necessários.
